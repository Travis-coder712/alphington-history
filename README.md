# A History of Alphington

A deep local-history PWA about Alphington, Victoria (postcode 3078), on the Birrarung/Yarra River in Melbourne's inner north-east.

Sixteen sections covering: Wurundjeri Country; the 1840 Crown portions and the naming of the suburb; South Alphington; a planning & heritage **deep-dive on Roemer Crescent** (Heritage Overlay HO362) with original schematic maps; **Runnymeade and the Renn family gardens**; grand homes and estates; the **Australian Paper Mill → YarraBend**; the **Grandview Hotel** legend; **Parkside Football Club**; the local bowls/soccer/cricket/netball clubs; **Alphington Primary School**; neighbouring Fairfield, Heidelberg and Clifton Hill; notable people; a timeline; and full sources.

Every substantive claim is tagged — **documented**, **local legend**, or **unverified** — so record and memory stay distinct. Sources are cited per section.

## Tech

Static HTML/CSS/JS, no build step. Installable PWA (manifest + service worker, offline-capable). Two hand-drawn SVG schematic maps (Crown portions; Roemer Crescent curve with per-house heritage gradings).

## Run locally

```bash
python3 -m http.server 5197
```
Then open http://localhost:5197/

Icons are generated by `make_icons.py` (requires Pillow).
